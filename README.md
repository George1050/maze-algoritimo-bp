# maze-algoritimo-bp
Implementação em c do algoritmo de busca em profundidade usando um labirinto qualquer.

Para a execução:

1 - informe as dimensões do labirinto(5 5)

2 - Escreva o labirinto(#0####000####0##000####0#).
obs: escreva em uma unica linha;

5 5
